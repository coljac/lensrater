# Lensrater

Application for quickly classifying images.

## Usage

|  |  |
|---------------|--------|
| Next image | right, j |
| Prev image | left, k |
| Forward/back 10 images | pgup/pgdn |
| First/last image | Home/End |
| Save and quit | Esc |
| Classify 0 | 0, ~ |
| Classify 1 | 1 |
| Classify 2 | 2 |
| Classify 3 | 3 |
| Increase score | Space |
